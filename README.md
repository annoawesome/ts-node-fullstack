# ts-node-fullstack

My GitHub template for getting started with full-stack NodeJS with TypeScript. Includes the following things preconfigured:

- ESLint
- Prettier
- Vite
- VS Code (specifically, `tasks.json`)

## Getting Started

It is highly recommended to use Dev Containers. To open the project in a Dev Container, install the VS Code extension and press `F1` on your keyboard to run `Dev Containers: Open Folder in Container`. Then select the `Node.js + TypeScript` template. Make sure to include any optional features such as databases when asked. For more information: read the [guide](https://code.visualstudio.com/docs/devcontainers/create-dev-container) or visit [the official website](https://containers.dev/).

The template may be slightly out of date. Run `npm i -D` in both the project root and `src/public` to update and install packages. Additionally, overwrite `package.json` and change the license to whatever you want.

Frameworks like Express.js and React are not included by default.
